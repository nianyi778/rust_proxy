# 对外发布素材（Markdown）

这里放的是可直接复制发布的文案（Markdown）。你可以按渠道选择对应文件。

## 文件列表

- 掘金长文：`docs/press/juejin.md`
- 推特/微博线程：`docs/press/twitter-thread.md`
- 个人博客版（技术复盘）：`docs/press/blog.md`

## 图片占位

正文里用相对路径留了图片占位。你只需要把图片放到：

- `docs/press/images/architecture.png`
- `docs/press/images/demo.png`

发布到外部平台时：

- 平台支持上传图片：先上传图片，再把 `![...](images/xxx.png)` 替换成平台图片 URL。
- 用 GitHub 直链：把图片推到仓库后，取 Raw 链接替换。
